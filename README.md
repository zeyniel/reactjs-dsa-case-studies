# 🚀 Data Structures and Algorithms: Case Studies

A comprehensive React-based visualization platform designed to demonstrate core computer science concepts through interactive modules. This project features a clean, modern interface for exploring algorithms and data structures in real-time.

## 🛠️ Featured Case Studies
* **TicTacToe:** Interactive game logic and state management.
* **Stacks & Queues:** Visual demonstrations of linear data structure operations.
* **Binary Search Tree (BST):** Node insertion and searching logic.
* **Binary Tree Traversal:** Visualizing In-order, Pre-order, and Post-order paths.
* **Tower of Hanoi:** A step-by-step visualization of recursive problem-solving.
* **Sorting Algorithms:** Animating various sorting techniques for better conceptual understanding.

## 💻 Tech Stack
* **Framework:** ReactJS (Functional Components & Hooks)
* **Build Tool:** Vite
* **Routing:** React Router (HashRouter for GitHub Pages support)
* **Styling:** Custom CSS with Euclid Circular typography
